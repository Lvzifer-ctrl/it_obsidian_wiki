### Was ist eine UI?
Die **Benutzerschnittstelle (User Interface)** ist die Gesamtheit aller Elemente, über die ein Nutzer mit einem Computer, Smartphone oder System interagiert.
Dazu gehören grafische Oberflächen, Tasten, Menüs, Touchgesten und mehr.

Sie bildet die sichtbare Brücke zwischen Nutzer und Technologie.
***
### Arten der Benutzerschnittstelle
- Grafische Benutzeroberflächen
- Touch-Oberflächen
- Sprachsteuerung
- Kommandozeilen
- Menübasierte Systeme
